# Minions Memory

Persistent cross-session memory, recalled facts, and agent knowledge base

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-memory/sdk`) — Core library for Node.js
- **Python SDK** (`minions-memory`) — Python bindings
- **CLI Tool** (`@minions-memory/cli`) — Command-line interface
- **Web App** — Interactive playground at [memory.minions.wtf](https://memory.minions.wtf)
