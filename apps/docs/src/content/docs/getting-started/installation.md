---
title: Installation
description: Install Minions Memory packages
---

## TypeScript / Node.js

```bash
npm install @minions-memory/sdk minions-sdk
```

## Python

```bash
pip install minions-memory
```

## CLI

```bash
npm install -g @minions-memory/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
